NLP Preprocessing Engine
This project implements a simple NLP preprocessing pipeline to clean noisy real-world text data.

Features
Converts text to lowercase
Removes URLs, emails, and numbers
Handles repeated characters
Removes special characters and extra spaces
Tokenization and basic filtering
File
NLP_Task1.ipynb – main notebook containing implementation and results
Usage
Run the notebook in Google Colab or Jupyter Notebook to see preprocessing, token analysis, and frequency analysis.
